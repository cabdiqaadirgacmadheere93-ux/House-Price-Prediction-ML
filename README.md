# House Price Prediction Project
This project uses **Multiple Linear Regression** to predict house prices based on features like square footage, number of bedrooms, and year built.

## Project Structure
- `house_price_regression_dataset.csv`: The dataset used for training.
- `House_Price_Project.ipynb`: The main Python code (Jupyter Notebook).

## Requirements
- Python 3.x
- Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn
